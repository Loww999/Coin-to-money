# Coin to Money

**Coin to Money** is a PocketMine-MP plugin that allows players to **convert their coins into money** in your server. This plugin is designed to simplify the economy and reward system in Minecraft Pocket Edition.

## Features

* Convert player coins into money.
* Configurable conversion rates.
